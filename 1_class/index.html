Variables

Depende donde la pongamos es donde la puedo usar

var cuando lo pones el codigo genera la definicion de variable al principio de todo. Se llama Hoisting

let cuando lo pones el codigo genera la definicion solo en el bloque de codigo en donde se define

const genera una variable global el cual no se puede re definir y esta accesible en todos lados

Break y Continue

Podemos cortar o saltear el ciclo

                
                    const limite = 10
                    const corte = 10
                    for(let i = 0; i < limite; i++) {
                      console.log('Soy ' + i);
                      if(i === corte) {
                        break;
                      }
                      console.log('Sobrevivi porque no soy ' + corte);
                    }
                
            
                
                    const limite = 10
                    const corte = 10
                    for(let i = 0; i < limite; i++) {
                      console.log('Soy ' + i);
                      if(i === corte) {
                        continue;
                      }
                      console.log('Sobrevivi porque no soy ' + corte);
                    }
